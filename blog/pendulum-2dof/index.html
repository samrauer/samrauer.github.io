Dynamics of a 2-DOF pendulum

This page derives the equations of motion for a 2-degree-of-freedom pendulum with variable rope length and a fixed pivot point.

Notation guide

  • θ\thetaθ: pendulum azimuthal angle
  • ϕ\phiϕ: pendulum nadir angle
  • ℓ\ellℓ: length of pendulum arm
  • bbb: pendulum bob point
  • aaa: pivot point
  • mmm: mass of pendulum bob
  • FA\rmF_\rmAFA​: inertial frame
  • FB\rmF_\rmBFB​: frame aligned with pendulum azimuth angle
  • FC\rmF_\rmCFC​: frame align with pendulum azimuth and nadir angle
  • (ı^(⋅),ȷ^(⋅),k^(⋅))(\ihat_{(\cdot)},\jhat_{(\cdot)},\khat_{(\cdot)})(^(⋅)​,^​(⋅)​,k^(⋅)​): the basis vectors of F(⋅)\rmF_{(\cdot)}F(⋅)​
  • ω⇀B/A\vect{\omega}_{\rmB/\rmA}ω⇀B/A​: angular velocity of FB\rmF_\rmBFB​ relative to FA\rmF_\rmAFA​
  • ω⇀C/A\vect{\omega}_{\rmC/\rmA}ω⇀C/A​: angular velocity of FC\rmF_\rmCFC​ relative to FA\rmF_\rmAFA​
  • ω⇀C/B\vect{\omega}_{\rmC/\rmB}ω⇀C/B​: angular velocity of FC\rmF_\rmCFC​ relative to FB\rmF_\rmBFB​
  • r⇀b/a\vect{r}_{b/a}r⇀b/a​: position of bbb relative to aaa
  • v⇀b/a/A\vect{v}_{b/a/\rmA}v⇀b/a/A​: velocity of bbb relative to aaa with respect to FA\rmF_\rmAFA​
  • Tb/a/AT_{b/a/\rmA}Tb/a/A​: kinetic energy of the pendulum mass relative to aaa with respect to FA\rmF_\rmAFA​, as a function of generalized coordinates
  • Vb/aV_{b/a}Vb/a​: potential energy of the pendulum mass relative to aaa, as a function of generalized coordinates
  • ggg: gravitational acceleration magnitude
  • g⇀\vect{g}g⇀​: gravitational acceleration vector
  • qqq: generalized coordinates vector
  • Lb/a/AL_{b/a/\rmA}Lb/a/A​: Lagrangian of the pendulum mass system
  • dt(⋅)d_t (\cdot)dt​(⋅): derivative with respect to time
  • ∂q(⋅)\partial_q (\cdot)∂q​(⋅) / ∂q˙(⋅)\partial_{\dot{q}} (\cdot)∂q˙​​(⋅): partial derivative with respect to qqq / q˙\dot{q}q˙​

Dynamics derivation

Let’s define our system. The frame setup is

FA⟶3θFB⟶1ϕFC\rmF_\rmA \quad \overset{\theta}{ \underset{3}{\longrightarrow} } \quad \rmF_\rmB \quad \overset{\phi}{ \underset{1}{\longrightarrow} } \quad \rmF_\rmCFA​3⟶​θ​FB​1⟶​ϕ​FC​

where

  • FA\rmF_\rmAFA​ is an inertial frame
  • FB\rmF_\rmBFB​ is obtained by rotating FA\rmF_\rmAFA​ about k^A\khat_\rmAk^A​ by θ\thetaθ such that ȷ^B\jhat_\rmB^​B​ is aligned with the pendulum’s azimuthal angle
  • FC\rmF_\rmCFC​ is obtained by rotating FB\rmF_\rmBFB​ about ı^B\ihat_\rmB^B​ by ϕ\phiϕ such that k^C\khat_\rmCk^C​ is aligned with the pendulum’s nadir angle

The frame angular velocities are then

ω⇀B/A=θ˙k^A=θ˙k^B\vect{\omega}_{\rmB/\rmA} = \dot{\theta} \khat_\rmA = \dot{\theta} \khat_\rmBω⇀B/A​=θ˙k^A​=θ˙k^B​ ω⇀C/B=ϕ˙ı^B=ϕ˙ı^C\vect{\omega}_{\rmC/\rmB} = \dot{\phi} \ihat_\rmB = \dot{\phi} \ihat_\rmCω⇀C/B​=ϕ˙​^B​=ϕ˙​^C​

The fixed pivot point is denoted aaa and the position of the pendulum mass is denoted bbb. The position of the mass relative to aaa is

r⇀b/a=−ℓk^C.\vect{r}_{b/a} = -\ell \khat_\rmC.r⇀b/a​=−ℓk^C​.

The velocity is obtained by taking the derivative of r⇀b/a\vect{r}_{b/a}r⇀b/a​ with respect to an inertial frame and using the transport theorem

v⇀b/a/A=r⇀A∙b/a=−ℓ˙k^C−ℓ(ω⇀C/A×k^C)=−ℓ˙k^C−ℓ((ω⇀C/B+ω⇀B/A)×k^C)=−ℓ˙k^C−ℓ((ϕ˙ı^C+θ˙k^B)×k^C)\begin{align*} \vect{v}_{b/a/\rmA} = \framedota{\vect{r}}_{b/a} &= -\dot{\ell} \khat_\rmC - \ell (\vect{\omega}_{\rmC/\rmA} \times \khat_\rmC) \\ &= -\dot{\ell} \khat_\rmC - \ell \left( (\vect{\omega}_{\rmC/\rmB} + \vect{\omega}_{\rmB/\rmA}) \times \khat_\rmC \right) \\ &= -\dot{\ell} \khat_\rmC - \ell \left( (\dot{\phi} \ihat_\rmC + \dot{\theta} \khat_\rmB) \times \khat_\rmC \right) \\ \end{align*}v⇀b/a/A​=r⇀A∙b/a​​=−ℓ˙k^C​−ℓ(ω⇀C/A​×k^C​)=−ℓ˙k^C​−ℓ((ω⇀C/B​+ω⇀B/A​)×k^C​)=−ℓ˙k^C​−ℓ((ϕ˙​^C​+θ˙k^B​)×k^C​)​

k^B\khat_\rmBk^B​ expressed with the FC\rmF_\rmCFC​ basis vectors is

k^B=sin⁡(ϕ)ȷ^C+cos⁡(ϕ)k^C\khat_\rmB = \sin(\phi) \jhat_\rmC + \cos(\phi) \khat_\rmCk^B​=sin(ϕ)^​C​+cos(ϕ)k^C​

so

v⇀b/a/A=−ℓ˙k^C−ℓ((ϕ˙ı^C+θ˙k^B)×k^C)=−ℓ˙k^C−ℓ((ϕ˙ı^C+θ˙sin⁡(ϕ)ȷ^C+θ˙cos⁡(ϕ)k^C)×k^C)=−ℓ˙k^C−ℓ(−ϕ˙ȷ^C+θ˙sin⁡(ϕ)ı^C)=−ℓθ˙sin⁡(ϕ)ı^C+ℓϕ˙ȷ^C−ℓ˙k^C\begin{align*} \vect{v}_{b/a/\rmA} &= -\dot{\ell} \khat_\rmC - \ell \left( (\dot{\phi} \ihat_\rmC + \dot{\theta} \khat_\rmB) \times \khat_\rmC \right) \\ &= -\dot{\ell} \khat_\rmC - \ell \left( (\dot{\phi} \ihat_\rmC + \dot{\theta} \sin(\phi) \jhat_\rmC + \dot{\theta} \cos(\phi) \khat_\rmC) \times \khat_\rmC \right) \\ &= -\dot{\ell} \khat_\rmC - \ell \left( -\dot{\phi} \jhat_\rmC + \dot{\theta}\sin(\phi) \ihat_\rmC \right) \\ &= -\ell \dot{\theta} \sin(\phi) \ihat_\rmC + \ell \dot{\phi} \jhat_\rmC - \dot{\ell} \khat_\rmC \end{align*}v⇀b/a/A​​=−ℓ˙k^C​−ℓ((ϕ˙​^C​+θ˙k^B​)×k^C​)=−ℓ˙k^C​−ℓ((ϕ˙​^C​+θ˙sin(ϕ)^​C​+θ˙cos(ϕ)k^C​)×k^C​)=−ℓ˙k^C​−ℓ(−ϕ˙​^​C​+θ˙sin(ϕ)^C​)=−ℓθ˙sin(ϕ)^C​+ℓϕ˙​^​C​−ℓ˙k^C​​

The kinetic energy of the system is

Tb/a/A=12mv⇀b/a/A′v⇀b/a/A=12m((−ℓθ˙sin⁡(ϕ))2+(ℓϕ˙)2+(−ℓ˙)2)=12m(ℓ2θ˙2sin⁡2(ϕ)+ℓ2ϕ˙2+ℓ˙2)\begin{align*} T_{b/a/\rmA} &= \frac{1}{2} m \vect{v}_{b/a/\rmA}' \vect{v}_{b/a/\rmA} \\ &= \frac{1}{2} m \left( (-\ell \dot{\theta} \sin(\phi))^2 + (\ell \dot{\phi})^2 + (- \dot{\ell})^2 \right) \\ &= \frac{1}{2} m \left( \ell^2 \dot{\theta}^2 \sin^2(\phi) + \ell^2 \dot{\phi}^2 + \dot{\ell}^2 \right) \end{align*}Tb/a/A​​=21​mv⇀b/a/A′​v⇀b/a/A​=21​m((−ℓθ˙sin(ϕ))2+(ℓϕ˙​)2+(−ℓ˙)2)=21​m(ℓ2θ˙2sin2(ϕ)+ℓ2ϕ˙​2+ℓ˙2)​

The potential energy of the system is

Vb/a=−mg⇀′(r⇀b/a)=−m(−gk^B)′(−ℓk^C)=−mgℓk^B′(−sin⁡(ϕ)ȷ^B+cos⁡(ϕ)k^B)=−mgℓcos⁡(ϕ)\begin{align*} V_{b/a} &= -m \vect{g}' ( \vect{r}_{b/a} ) \\ &= -m (-g \khat_\rmB)' (-\ell \khat_\rmC) \\ &= -m g \ell \khat_\rmB' \left(-\sin(\phi) \jhat_\rmB + \cos(\phi) \khat_\rmB \right) \\ &= -m g \ell \cos(\phi) \end{align*}Vb/a​​=−mg⇀​′(r⇀b/a​)=−m(−gk^B​)′(−ℓk^C​)=−mgℓk^B′​(−sin(ϕ)^​B​+cos(ϕ)k^B​)=−mgℓcos(ϕ)​

Combining the kinetic and potential energy of the system we get the lagrangian

Lb/a/A≡Tb/a/A−Vb/a=12m(ℓ2θ˙2sin⁡2(ϕ)+ℓ2ϕ˙2+ℓ˙2)+mgℓcos⁡(ϕ)\begin{align*} L_{b/a/\rmA} &\equiv T_{b/a/\rmA} - V_{b/a} \\ &= \frac{1}{2} m \left( \ell^2 \dot{\theta}^2 \sin^2(\phi) + \ell^2 \dot{\phi}^2 + \dot{\ell}^2 \right) + m g \ell \cos(\phi) \end{align*}Lb/a/A​​≡Tb/a/A​−Vb/a​=21​m(ℓ2θ˙2sin2(ϕ)+ℓ2ϕ˙​2+ℓ˙2)+mgℓcos(ϕ)​

Define qqq as

q=[θϕ]q = \begin{bmatrix} \theta \\ \phi \end{bmatrix}q=[θϕ​]

There are no non-conservative, external, or dissipative forces, so Q=0Q=0Q=0. Using the the Euler-Lagrange equation, we can get the equations of motion for the pendulum system

ddt∂∂q˙Lb/a/A−∂∂qLb/a/A=Q=0\frac{d}{dt} \frac{\partial}{\partial \dot{q}} L_{b/a/\rmA} - \frac{\partial}{\partial q} L_{b/a/\rmA} = Q = 0dtd​∂q˙​∂​Lb/a/A​−∂q∂​Lb/a/A​=Q=0

For θ\thetaθ we get

dt∂θ˙L−∂θL=0dt(mθ˙ℓ2sin⁡2(ϕ))−(0)=0mθ¨ℓ2sin⁡2(ϕ)+2mθ˙ℓℓ˙sin⁡2(ϕ)+2mθ˙ℓ2sin⁡(ϕ)cos⁡(ϕ)ϕ˙=0θ¨ℓsin⁡(ϕ)+θ˙(2ℓ˙sin⁡(ϕ)+2ℓcos⁡(ϕ)ϕ˙)=0\begin{align*} d_t \partial_{\dot{\theta}} L - \partial_\theta L &= 0 \\ d_t \left( m \dot{\theta} \ell^2 \sin^2(\phi) \right) - (0) &= 0 \\ m \ddot{\theta} \ell^2 \sin^2(\phi) + 2 m \dot{\theta} \ell \dot{\ell} \sin^2(\phi) + 2 m \dot{\theta} \ell^2 \sin(\phi) \cos(\phi) \dot{\phi} &= 0 \\ \ddot{\theta} \ell \sin(\phi) + \dot{\theta} \left( 2 \dot{\ell} \sin(\phi) + 2 \ell \cos(\phi) \dot{\phi} \right) &= 0 \end{align*}dt​∂θ˙​L−∂θ​Ldt​(mθ˙ℓ2sin2(ϕ))−(0)mθ¨ℓ2sin2(ϕ)+2mθ˙ℓℓ˙sin2(ϕ)+2mθ˙ℓ2sin(ϕ)cos(ϕ)ϕ˙​θ¨ℓsin(ϕ)+θ˙(2ℓ˙sin(ϕ)+2ℓcos(ϕ)ϕ˙​)​=0=0=0=0​

For ϕ\phiϕ we get

dt∂ϕ˙L−∂ϕL=0dt(mϕ˙ℓ2)−(12m(2θ˙2ℓ2sin⁡(ϕ)cos⁡(ϕ))−mgℓsin⁡(ϕ))=0mℓ2ϕ¨+2mϕ˙ℓℓ˙−mθ˙2ℓ2sin⁡(ϕ)cos⁡(ϕ)+mgℓsin⁡(ϕ)=0ϕ¨ℓ+2ℓ˙ϕ˙−ℓθ˙2sin⁡(ϕ)cos⁡(ϕ)+gsin⁡(ϕ)=0\begin{align*} d_t \partial_{\dot{\phi}} L - \partial_\phi L &= 0 \\ d_t \left( m \dot{\phi} \ell^2 \right) - \left( \frac{1}{2}m \left(2 \dot{\theta}^2 \ell^2 \sin(\phi) \cos(\phi) \right) - m g \ell \sin(\phi) \right) &= 0 \\ m \ell^2 \ddot{\phi} + 2 m \dot{\phi} \ell \dot{\ell} - m \dot{\theta}^2 \ell^2 \sin(\phi) \cos(\phi) + m g \ell \sin(\phi) &= 0 \\ \ddot{\phi} \ell + 2 \dot{\ell} \dot{\phi} - \ell \dot{\theta}^2 \sin(\phi) \cos(\phi) + g \sin(\phi) &= 0 \end{align*}dt​∂ϕ˙​​L−∂ϕ​Ldt​(mϕ˙​ℓ2)−(21​m(2θ˙2ℓ2sin(ϕ)cos(ϕ))−mgℓsin(ϕ))mℓ2ϕ¨​+2mϕ˙​ℓℓ˙−mθ˙2ℓ2sin(ϕ)cos(ϕ)+mgℓsin(ϕ)ϕ¨​ℓ+2ℓ˙ϕ˙​−ℓθ˙2sin(ϕ)cos(ϕ)+gsin(ϕ)​=0=0=0=0​

We now have the equations of motion for our 2 degree of freedom pendulum!

θ¨ℓsin⁡(ϕ)+θ˙(2ℓ˙sin⁡(ϕ)+2ℓcos⁡(ϕ)ϕ˙)=0\ddot{\theta} \ell \sin(\phi) + \dot{\theta} \left( 2 \dot{\ell} \sin(\phi) + 2 \ell \cos(\phi) \dot{\phi} \right) = 0θ¨ℓsin(ϕ)+θ˙(2ℓ˙sin(ϕ)+2ℓcos(ϕ)ϕ˙​)=0 ϕ¨ℓ+2ℓ˙ϕ˙−ℓθ˙2sin⁡(ϕ)cos⁡(ϕ)+gsin⁡(ϕ)=0\ddot{\phi} \ell + 2 \dot{\ell} \dot{\phi} - \ell \dot{\theta}^2 \sin(\phi) \cos(\phi) + g \sin(\phi) = 0ϕ¨​ℓ+2ℓ˙ϕ˙​−ℓθ˙2sin(ϕ)cos(ϕ)+gsin(ϕ)=0

Note that ℓ\ellℓ is a prescribed length, so no ℓ¨\ddot{\ell}ℓ¨ shows up. If the length of the rope doesn’t change, then the equations of motion reduce to

θ¨sin⁡(ϕ)+2θ˙ϕ˙cos⁡(ϕ)=0\ddot{\theta} \sin(\phi) + 2 \dot{\theta} \dot{\phi} \cos(\phi) = 0θ¨sin(ϕ)+2θ˙ϕ˙​cos(ϕ)=0 ϕ¨−θ˙2sin⁡(ϕ)cos⁡(ϕ)+gℓsin⁡(ϕ)=0\ddot{\phi} - \dot{\theta}^2 \sin(\phi) \cos(\phi) + \frac{g}{\ell} \sin(\phi) = 0ϕ¨​−θ˙2sin(ϕ)cos(ϕ)+ℓg​sin(ϕ)=0

If θ\thetaθ is fixed to θ=0\theta = 0θ=0, so we reduce the pendulum down to 1 DOF, the equations of motion reduce to the classic pendulum equation

ϕ¨+gℓsin⁡(ϕ)=0\ddot{\phi} + \frac{g}{\ell} \sin(\phi) = 0ϕ¨​+ℓg​sin(ϕ)=0

Simulation

We can integrate an initial pendulum state with a varying length rope using ODE45 or a similar integrator. Below is an example in MATLAB. Try it out yourself!

% Simulation of 2-DOF pendulum with prescribed rope length

% Parameters
g = 9.81;                              % gravity [m/s^2]
ell_fun = @(t) 1 + 0.2*sin(0.5*t);     % rope length function ell(t)
ell_dot_fun = @(t) 0.2*0.5*cos(0.5*t); % rope length derivative elldot(t)

% Initial conditions: [theta, theta_dot, phi, phi_dot]
theta0     = 0;     % azimuth [rad]
theta_dot0 = 0.02;  % azimuth rate [rad/s]
phi0       = pi/4;  % nadir [rad]
phi_dot0   = 0;     % nadir rate [rad/s]

y0 = [theta0; theta_dot0; phi0; phi_dot0];

% Time span
tspan = [0, 5];

% ODE function
odefun = @(t, y) pendulumODE(t, y, ell_fun, ell_dot_fun, g);

% Solve
[t, y] = ode45(odefun, tspan, y0);

theta     = y(:,1);
theta_dot = y(:,2);
phi       = y(:,3);
phi_dot   = y(:,4);

% Plot results
figure(1);
clf(1);
subplot(3,1,1);
plot(t, theta, "LineWidth", 1.5);
ylabel("$\theta$ [rad]", Interpreter="latex");

subplot(3,1,2);
plot(t, phi, "LineWidth", 1.5);
ylabel("$\phi$ [rad]", Interpreter="latex");

subplot(3,1,3);
plot(t, ell_fun(t), "LineWidth", 1.5);
ylabel("$\ell(t)$ [m]", Interpreter="latex");
xlabel("Time [s]", Interpreter="latex");

sgtitle("2-DOF pendulum with changing rope length");

%% --------------------------------------------------------
function dydt = pendulumODE(t, y, ell_fun, ell_dot_fun, g)
    theta     = y(1);
    theta_dot = y(2);
    phi       = y(3);
    phi_dot   = y(4);

    ell     = ell_fun(t);
    ell_dot = ell_dot_fun(t);

    % Eqn 1: theta ddot
    theta_ddot = -(theta_dot*(2*ell_dot*sin(phi) + 2*ell*cos(phi)*phi_dot)) ...
                 / (ell*sin(phi));

    % Eqn 2: phi ddot
    phi_ddot = (-2*ell*ell_dot*phi_dot + ell^2*theta_dot^2*sin(phi)*cos(phi) ...
                - g*ell*sin(phi)) / (ell^2);

    dydt = [theta_dot;
            theta_ddot;
            phi_dot;
            phi_ddot];
end
